# Expense Tracker – Frontend

This repository contains the frontend of the Expense Tracker application.
The frontend is built using HTML, CSS, and Vanilla JavaScript and communicates with a locally running backend API.

## Features
- User Registration
- User Login using JWT authentication
- Protected Dashboard
- Add, view, and delete expenses
- Logout functionality

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- Fetch API

## Project Structure
index.html        – Login & Register page  
dashboard.html    – Dashboard page  
auth.js           – Authentication logic (register & login)  
style.css         – Styling  

## How to Run Locally
1. Clone the repository:
git clone https://github.com/meeta163/expense-tracker-frontend.git


2. Open the project folder.

3. Open `index.html` using Live Server or directly in a browser.
4. or npx serve

> Make sure the backend server is running before using the application.

## Backend API
The frontend connects to a backend running locally at:
http://localhost:5000


## Tested Scenarios

- Access protected dashboard
- Add expenses
